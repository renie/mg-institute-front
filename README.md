## Dependencies

This project requires just **NodeJS (version >= 14.3)**.

## Setup

Install deps:
```
npm i
```

## Build

This will generate a `dist` directory with build files in it.
If you are using another webserver to serve it, you should point root directory this `dist`.
```
npm run build
```
